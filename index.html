Quick Deployment (5 Minutes)

  1. Prepare Snowflake Environment

    Open SNOWSIGHT_SETUP.sql from this project and run it in Snowsight SQL Worksheet. This creates the database, schema, and warehouse.

    USE ROLE ACCOUNTADMIN;
CREATE DATABASE IF NOT EXISTS STREAMLIT_DB;
CREATE SCHEMA IF NOT EXISTS STREAMLIT_DB.CREDIT_ANALYZER;
CREATE WAREHOUSE IF NOT EXISTS STREAMLIT_WH 
    WITH WAREHOUSE_SIZE='X-SMALL' AUTO_SUSPEND=60;

  2. Create Streamlit App in Snowsight

    Navigate to: Projects ‚Üí Streamlit ‚Üí + Streamlit App

    Configure:

    • App name: Credit Usage Analyzer
    • Warehouse: STREAMLIT_WH
    • Location: STREAMLIT_DB.CREDIT_ANALYZER

  3. Deploy the Code

    Open streamlit_app.py from this project, copy ALL the code, and paste it into Snowsight's Streamlit editor. Click Run.

  4. View Your Dashboard

    The app will load in Snowsight with credit usage analysis. Share with your team using the Share button.

Why Snowsight-Only?

This application uses Snowflake-specific features that only work in Snowsight:

  • get_active_session() - Automatic Snowflake authentication
  • SNOWFLAKE.ACCOUNT_USAGE views - System metadata tables
  • ACCOUNTADMIN privileges - Access to query history
